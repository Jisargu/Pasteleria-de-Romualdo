	GIT
	
	¿Cuál es la diferencia entre git y GitHUB?
Git es un controlador de versiones y github es un repositorio en la nube

	¿En cuantos y cuáles estados puede estar un archivo?
En cuatro: Working Directory ->(git add)-> Staging Area -> (git commit)-> Repository ->(git push)-> Remote

	¿Para qué sirve el comando "git add ." ?
Sirve para agregar todos los archivos a la Staging Area.

	¿Cuáles son los 4 datos mas relevantes a la hora de hacer un commit?

	Jenkins
	
	¿Cuál es el atajo (ruta relativa) para el directorio home de un usuario de ubuntu?
cd
----Otra forma----
cd ~

	¿Cómo se instala Jenkins en un sistema linux?
Actualizar los repositorios:
wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -

Añadir entrada a /etc/apt/sources:
deb https://pkg.jenkins.io/debian-stable binary/

Actualizar los indexes, instalar JAVA SDK y Jenkins (en ese orden):
sudo apt-get update
sudo apt install openjdk-8-jdk
sudo apt-get install jenkins
